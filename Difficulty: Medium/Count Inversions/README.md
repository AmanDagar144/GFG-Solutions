<h2><a href="https://www.geeksforgeeks.org/problems/inversion-of-array-1587115620/1">Count Inversions</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO" style="user-select: text;"><p style="user-select: text;"><span style="font-size: 14pt; user-select: text;">Given an array of integers <strong style="user-select: text;">arr[]</strong>. Find the <strong style="user-select: text;">Inversion Count</strong> in the array.<br style="user-select: text;">T</span><span style="font-size: 14pt; user-select: text;">wo elements arr[i] and arr[j] form an inversion if <strong style="user-select: text;">arr[i] &gt; arr[j]</strong> and <strong style="user-select: text;">i &lt; j</strong>.</span></p>
<blockquote style="user-select: text;">
<p style="user-select: text;"><span style="font-size: 14pt; user-select: text;"><em style="user-select: text;"><strong style="user-select: text;">Inversion Count</strong>: </em>For an array, inversion count indicates how far (or close) the array is from being sorted. If the array is already sorted then the inversion count is 0. <br style="user-select: text;">If an array is sorted in the reverse order then the inversion count is the maximum.&nbsp;</span></p>
</blockquote>
<p style="user-select: text;"><span style="font-size: 14pt; user-select: text;"><strong style="user-select: text;">Examples:</strong></span></p>
<pre style="user-select: text;"><span style="font-size: 14pt; user-select: text;"><strong style="user-select: text;">Input</strong>: arr[] = [2, 4, 1, 3, 5]<br style="user-select: text;"><strong style="user-select: text;">Output</strong>: 3
<strong style="user-select: text;">Explanation</strong>: The sequence 2, 4, 1, 3, 5 has three inversions (2, 1), (4, 1), (4, 3).</span></pre>
<pre style="user-select: text;"><span style="font-size: 14pt; user-select: text;"><strong style="user-select: text;">Input</strong>: arr[] = [2, 3, 4, 5, 6]<br style="user-select: text;"><strong style="user-select: text;">Output</strong>: 0
<strong style="user-select: text;">Explanation</strong>: As the sequence is already sorted so there is no inversion count.</span></pre>
<pre style="user-select: text;"><span style="font-size: 14pt; user-select: text;"><strong style="user-select: text;">Input</strong>: arr[] = [10, 10, 10]<br style="user-select: text;"><strong style="user-select: text;">Output</strong>: 0
<strong style="user-select: text;">Explanation</strong>: As all the elements of array are same, so there is no inversion count.</span></pre>
