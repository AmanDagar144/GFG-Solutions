<h2><a href="https://www.geeksforgeeks.org/problems/peak-element/1">Peak element</a></h2><h3>Difficulty Level : Difficulty: Basic</h3><hr><div class="problems_problem_content__Xm_eO" style="user-select: text;"><p style="user-select: text;"><span style="font-size: 14pt; user-select: text;">Given an array <strong style="user-select: text;">arr[]&nbsp;</strong>where no two adjacent elements are same, find the <strong style="user-select: text;">index </strong>of a <strong style="user-select: text;">peak </strong>element. An element is considered to be a <strong style="user-select: text;">peak</strong> if it is greater than its adjacent elements (if they exist). If there are multiple peak elements, return index of any one of them.&nbsp;</span><span style="font-size: 18.6667px; user-select: text;">The output will be&nbsp;</span><strong style="font-size: 18.6667px; user-select: text;">"true"</strong><span style="font-size: 18.6667px; user-select: text;">&nbsp;if the index returned by your function is correct; otherwise, it will be "</span><strong style="font-size: 18.6667px; user-select: text;">false"</strong><span style="font-size: 18.6667px; user-select: text;">.</span></p>
<p style="user-select: text;"><span style="font-size: 14pt; user-select: text;">Note: Consider the element <strong style="user-select: text;">before </strong>the <strong style="user-select: text;">first </strong>element and the element <strong style="user-select: text;">after </strong>the <strong style="user-select: text;">last </strong>element to be <strong style="user-select: text;">negative infinity</strong>.</span></p>
<p style="user-select: text;"><span style="font-size: 14pt; user-select: text;"><strong style="user-select: text;">Examples :<br style="user-select: text;"></strong></span></p>
<pre style="user-select: text;"><span style="font-size: 14pt; user-select: text;"><strong style="user-select: text;">Input: </strong>arr = [1, 2, 4, 5, 7, 8, 3]
<strong style="user-select: text;">Output:</strong> true
<strong style="user-select: text;">Explanation: </strong>arr[5] = 8 is a peak element because arr[4] &lt; arr[5] &gt; arr[6].</span></pre>
<pre style="user-select: text;"><span style="font-size: 14pt; user-select: text;"><strong style="user-select: text;">Input: </strong>arr = [10, 20, 15, 2, 23, 90, 80]
<strong style="user-select: text;">Output: </strong>true<strong style="user-select: text;">
Explanation: </strong>arr[1] = 20 and arr[5] = 90 are peak elements because arr[0] &lt; arr[1] &gt; arr[2] and arr[4] &lt; arr[5] &gt; arr[6]. <br style="user-select: text;"></span></pre>
<pre style="user-select: text;"><span style="font-size: 14pt; user-select: text;"><strong style="user-select: text;">Input: </strong>arr = [1, 2, 3]
<strong style="user-select: text;">Output: </strong>true<strong style="user-select: text;">
Explanation: </strong>arr[2] is a peak element because arr[1] &lt; arr[2] and arr[2] is the last element, so it has negative infinity to its right.</span></pre>
